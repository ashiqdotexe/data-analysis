## ğŸ“ Dataset Description
The dataset contains information about customers, including:
- **ID**: Unique identifier for each customer.
- **Marital Status**: Single or Married.
- **Gender**: Male or Female.
- **Income**: Annual income of the customer.
- **Children**: Number of children.
- **Education**: Education level (e.g., Bachelors, Partial College).
- **Occupation**: Professional, Skilled Manual, etc.
- **Home Owner**: Whether the customer owns a home (Yes/No).
- **Cars**: Number of cars owned.
- **Commute Distance**: Distance sourced for commute (e.g., 0-1 Miles).
- **Region**: Region of the customer (e.g., North America, Europe).
- **Age**: Age of the customer.
- **Purchased Bike**: Target variable indicating if a bike was purchased (Yes/No).
